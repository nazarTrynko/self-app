## SEO STRATEGY FRAMEWORK

### For Each Landing Page:

1. **Primary Keyword** ‚Äî One main target (e.g., "journaling prompts")
2. **Secondary Keywords** ‚Äî 3-5 related terms in subheadings
3. **Long-tail** ‚Äî FAQ section targeting questions
4. **Internal Links** ‚Äî Connect related pages (journaling ‚Üî reflection ‚Üî therapy)
5. **Meta Description** ‚Äî 155 chars, includes keyword, entices click
6. **Structured Data** ‚Äî HowTo schema for prompt suites, FAQ schema for tips

### Content Structure for SEO:

```
H1: [Primary Keyword] ‚Äî [Benefit/Hook]
H2: Why These Prompts Work
H2: The [Name] Prompt Suite
   H3: Prompt 1: [Name]
   H3: Prompt 2: [Name]
   ...
H2: Pro Tips for Better Results
H2: What You'll Discover
H2: FAQ (long-tail keywords)
```

- [ ] < 2s load time

- [ ] Proper heading hierarchy
- [ ] Alt text on images
