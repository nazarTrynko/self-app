The Content Prompt Suite

Run these prompts in sequence. Each excavates a different dimension of compelling content.

1

The Hook Hunter

Excavates: What makes your audience stop scrolling
Prompt 
I want to create content about [your topic] for [your audience].

Help me understand what makes them stop scrolling:
1. What problems are they trying to solve right now?
2. What questions keep them up at night?
3. What beliefs do they hold that I can challenge?
4. What patterns do they recognize that I can interrupt?
5. What emotions are they feeling that I can amplify or shift?
6. What do they secretly want but are afraid to admit?

Be specific. "Success" is too vague. "Escaping the 9-to-5 without going broke" is specific.
2

The Formulator

Excavates: Proven hook formulas that work
Prompt 
Based on what makes my audience stop scrolling [from Prompt 1], help me identify which hook formulas would work best:

1. **Curiosity Gap** — "The [unexpected thing] that [surprising outcome]"
2. **Pattern Interrupt** — "Everything you know about [topic] is wrong"
3. **Specificity** — "How I [specific action] and [specific result]"
4. **Contrarian** — "Why [common belief] is holding you back"
5. **Story Hook** — "I [did thing] and [unexpected result] happened"
6. **Question** — "What if [provocative question]?"
7. **Number** — "[Number] [things] that [outcome]"

For each formula, explain why it would work for my audience and give me a template I can fill in.
3

The Variator

Excavates: 20+ hook variations using formulas
Prompt 
Using the formulas from Prompt 2, generate 20+ hook variations for my content about [topic] for [audience].

Requirements:
- Use at least 5 different formulas
- Vary the tone (some bold, some subtle)
- Include specific numbers/details where relevant
- Make each hook feel fresh, not templated
- Optimize for [platform: LinkedIn/Twitter/etc]

Format each hook clearly and explain which formula it uses. Rank them by "stopping power" — which would make someone pause mid-scroll?
4

The Tester

Excavates: A/B test variations with reasoning
Prompt 
From the 20+ hooks in Prompt 3, help me create 3 A/B test pairs:

For each pair:
1. Hook A: [variation 1]
2. Hook B: [variation 2]
3. What's the key difference between them?
4. What hypothesis am I testing?
5. Which do you predict will perform better and why?

Then give me a framework for testing:
- What metric should I track?
- How long should I test?
- What would make me switch to the winner permanently?
5

The Optimizer

Excavates: Final optimized hook with explanation
Prompt 
Based on all the hooks we've generated, give me the FINAL optimized hook for [topic] on [platform].

Requirements:
- Maximum stopping power
- Authentic to my voice
- Optimized for [platform] best practices
- Includes a clear promise or question

Then explain:
1. Why this hook will work
2. What makes it better than the alternatives
3. What the first sentence of the content should be (to deliver on the hook)
4. How to adapt it if it doesn't perform

Make it ready to use.

Pro Tips

Use the Hook Generator

After running prompts 1-2, use the Hook Formula Engine above to instantly generate 20+ variations. Compare viral potential scores.

Test Everything

Never assume a hook will work. Test 3-5 variations and let data decide. What works for one audience might fail for another.

Platform Matters

LinkedIn hooks are different from Twitter hooks. Use the platform selector in the tool to optimize for your channel.

Iterate Fast

If a hook underperforms, go back to Prompt 3 and generate new variations. Don't stick with a weak hook out of pride.
