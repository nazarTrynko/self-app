### Visual Brand System

**Logo Concept:**
- Wordmark "BEST PROMPTS" 
- Or: "BP" monogram with distinctive typography
- Clean, modern, memorable

**Color Philosophy:**
- Dark mode default (easier on eyes for long reading)

- Decision = Red, LinkedIn = Orange, Journal = Warm Earth tones
- This prevents monotony across 50+ pages

- Display: Changes per chamber (creates variety)
- Body: Consistent across site (readability)
- Mono: For code/prompts (technical clarity)

**Photography/Imagery:**
- NO stock photos (especially not of people pointing at screens)
- YES abstract textures, gradients, custom graphics
- YES tool screenshots/outputs
- Illustrations only if distinctive (not generic)

### Pricing Psychology

**Principles:**

1. **Anchor High, Offer Value**
   - Premium packs at $29 make $9 packs feel like steals
   - Course at $299 makes $99 feel accessible

2. **One-Time Purchases Win Early**
   - Easier conversion than subscriptions
   - Lower commitment barrier
   - Can offer bundle discounts

3. **Freemium Balance: 80/20**
   - 80% of value stays free (builds trust)
   - 20% premium is for power users (makes money)

4. **Ethical Monetization**
   - Never paywall therapy/mental health prompts
   - Never create artificial scarcity
   - Genuinely more value in premium, not FOMO tactics

## CONTINGENCY PLANS

1. Are pages indexed? (Check GSC)
2. Are we ranking at all? (Even on page 2-3?)

4. Are target keywords too competitive?

**Responses:**
- Double down on long-tail keywords
- Create more link-worthy content
- Actively pursue backlinks
- Consider paid traffic test (small budget)

---

### Scenario: High Traffic, Low Conversions

**If traffic is good but email signup rate < 1%:**

**Analysis:**
1. Is value proposition clear?
2. Are CTAs prominent?
3. Is form friction too high?
4. Are we attracting wrong audience?

**Responses:**
- A/B test newsletter CTAs
- Add lead magnets (free prompt pack downloads)
- Improve copy around signup
- Segment audience, personalize offers

2. Is pricing right?

4. Is messaging clear?

## THE 10X QUESTION

**What would make this 10x better than anything that exists?**

**Current Answers:**

1. **Community Curation** — Let users submit prompts, vote on best
2. **Prompt Versioning** — Show evolution, let users remix
3. **Cross-Platform** — Same prompts work for ChatGPT, Claude, Gemini
4. **Use Case Stories** — Real examples of how prompts changed lives
5. **Prompt Analytics** — Track which prompts are most copied
6. **AI-Powered Customization** — (Future) Personalize prompts to user context

**The Non-Obvious Opportunity:**

Most prompt sites treat prompts as static text. We could treat them as **living documents** that evolve based on:
- User feedback ("this worked / didn't work")
- AI model changes (update prompts when GPT-5 launches)
- Community contributions (best variations surface)

This creates a **quality moat** competitors can't match.
