## THE PROMPT LIBRARY (Constellation Portal)

Open `index.html` in a browser.

### Structure

- `index.html`: Hub (Constellation Map)
- `chambers/*/index.html`: Chamber worlds
- `styles/`: shared + hub + chamber themes
- `scripts/`: shared utilities + hub + chamber tools

### Notes

- Pure HTML/CSS/JS, no frameworks
- Tools persist to `localStorage`
- Respects `prefers-reduced-motion`
