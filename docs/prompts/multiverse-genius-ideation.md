## UNIVERSE SELECTION CRITERIA

### Selection Framework

After exploring all universes, choose which one to create using these criteria:

#### 1. Novelty Score (0-10)
- How genuinely new is this universe's approach?
- Does it transcend or just combine existing methods?
- Would experts recognize this as a breakthrough?

#### 2. Feasibility Score (0-10)
- Can we actually create this universe from current reality?
- What's the path from here to there?
- What are the prerequisites?

#### 3. Value Potential (0-10)
- What's the potential impact of ideas from this universe?
- How valuable would these ideas be to users/market?
- What's the ROI of exploring this universe?

#### 4. Elegance Score (0-10)
- Is the universe's approach elegant and beautiful?
- Does it have unnecessary complexity?
- Is it the simplest path to breakthrough ideas?

#### 5. Sustainability (0-10)
- Can this universe sustain long-term ideation?
- Does it have built-in evolution capabilities?
- Will it remain valuable as context changes?

### Selection Formula

```
UNIVERSE_VALUE = (Novelty × 0.25) + (Feasibility × 0.20) + (Value_Potential × 0.25) + 
                 (Elegance × 0.15) + (Sustainability × 0.15)

Select universe with highest UNIVERSE_VALUE, but also consider:
- Feasibility must be ≥ 6 (otherwise it's fantasy, not reality)
- Novelty must be ≥ 7 (otherwise it's not truly breakthrough)
```
