# PromptCraft - GPT-4 Content Reference

**Use GPT-4 for:** User-facing content, nuanced explanations, quality copy

---

## Content That Needs Quality

### Landing Page Copy
- Hero headline: "Transform Your Prompts From Basic to Brilliant"
- Value prop: Educational + Useful in one interaction
- Social proof: "500+ on waitlist"

### Explanation Templates

Each transformation includes WHY it works:

```typescript
// Role Assignment explanation
"Adding an expert role helps the AI understand the perspective and expertise level expected."

// Format Specification explanation  
"Defining output structure reduces ambiguity and improves result quality."

// Constraint explanation
"Constraints focus the AI and prevent common issues in the output."
```

### Educational Content (Prompt DNA)

Topics to develop:
1. Anatomy of a Perfect Prompt

### Error Messages & UX Copy

```typescript
// Rate limit message
"Daily limit reached. Upgrade to Pro for unlimited transformations."

// Empty state

// Success feedback
"Transformation complete! Click any section to learn why it works."
```

## Quality Checklist for GPT-4 Tasks

- [ ] Is this user-facing?

- [ ] Will it impact conversions?
