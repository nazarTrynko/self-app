## The Synthesis Formula

```
SOURCE A (Theory/What)  +  SOURCE B (Technique/How)  =  SYNTHESIS (Novel System)
```

### Complementary Tension Matrix

Select sources with complementary tensions:

| Source A Role | Source B Role | Synthesis Power | Example |
|--------------|---------------|-----------------|---------|
| Theory (Why) | Technique (How) | ★★★★★ | McAdams + Bandler |
| Academic | Practitioner | ★★★★★ | Research + Application |
| Ancient Wisdom | Modern Science | ★★★★☆ | Stoicism + CBT |
| Cognitive | Somatic | ★★★★☆ | Mind + Body |
| Individual | Systemic | ★★★☆☆ | Personal + Context |

**The Golden Rule:** Source A tells you WHAT to change. Source B tells you HOW to change it.

## Phase 1: Source Selection

### Evaluation Criteria

Before committing to a synthesis, evaluate:

1. **Tension Quality**: What's the productive disagreement between sources?
2. **Bridge Potential**: How could they complement each other?
3. **Audience Clarity**: Who needs this synthesis specifically?
4. **Transformation Gap**: What becomes possible that neither enables alone?
5. **Differentiation**: What existing books cover similar ground?

### Source Selection Prompt

```markdown
I'm considering synthesizing these frameworks:

Source A: [Title/Author/Core Thesis]
Source B: [Title/Author/Core Thesis]

Evaluate:
1. What's the TENSION between them?
2. What's the BRIDGE that connects them?
3. Who NEEDS this synthesis? (Be specific)
4. What TRANSFORMATION becomes possible?
5. One-sentence synthesis thesis:

Kill signals:
- No clear tension = weak synthesis
- Vague audience = no distribution
- No novel transformation = why bother?
```

## Phase 4: Validation-First

### The Validation-First Model

**Never write a book before validating the synthesis.**

```
VALIDATION PIPELINE:
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│  1. SYNTHESIS HYPOTHESIS (1 day)                                │
│     └─▶ One-page thesis combining Source A + Source B           │
│                                                                  │
│  2. LANDING PAGE MVP (1-2 days)                                 │
│     └─▶ Hero + 1 Interactive Tool + Email Capture               │
│                                                                  │
│  3. VALIDATION GATES                                            │
│     └─▶ 100 email signups = PROCEED                             │
│     └─▶ 10 "would you pay $X?" = STRONG SIGNAL                  │
│     └─▶ Tool completion rate >60% = CONCEPT RESONATES           │
│                                                                  │
│  4. FULL BUILD (only after validation)                          │
│     └─▶ Complete landing + book + tools + app                   │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### Kill Gates

| Signal | Threshold | Action |
|--------|-----------|--------|
| Landing page bounce rate | >80% | Rethink hook |
| Email signup rate | <2% of visitors | Kill or pivot |
| Tool completion rate | <30% | Redesign tool |
| "Would pay $X" responses | <5 per 100 visitors | Kill concept |
| Negative feedback pattern | >20% | Address or kill |

## Legal & Ethics

### Intellectual Property

Synthesizing books requires:
1. **Attribution** — Always credit source authors
2. **Transformation** — Add novel value beyond sources
3. **Fair Use** — Commentary, criticism, education
4. **No Copying** — Concepts only, never text

**Safe Attribution Pattern:**
> "Building on McAdams' concept of redemptive sequences and integrating it with Bandler's state anchoring techniques, The Transition Rite introduces a novel protocol called..."

### Claims Calibration

| Claim Type | Acceptable | Dangerous |
|------------|------------|-----------|
| Personal experience | "This helped me..." | — |
| Framework | "This approach suggests..." | "This will cure..." |
| Research-backed | "Studies show..." (cited) | "Science proves..." |
| Transformational | "Many find that..." | "You will definitely..." |

### Required Disclaimers

Every book needs:
- Not therapy/medical advice disclaimer
- Results vary disclaimer
- Professional referral guidance
- Source attribution
