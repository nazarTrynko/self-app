## Overview

**The Amplified Mind** synthesizes Singularity Mode V2 (cognitive amplification protocol) with Critical Thinking frameworks (Paul & Elder, De Bono) to create a practical meta-cognitive operating system for calibrated, multi-perspectival reasoning.

## The 6-Layer Protocol

1. **Transmute (The Reframer)** — See the real problem
2. **Multiply (The Council)** — Gather expert perspectives  
3. **Evolve (The Challenger)** — Steel-man opposition
4. **Crystallize (The Pattern Finder)** — Extract principles
5. **Transcend (The Transcender)** — Complete the recursion
6. **Emerge (The Synthesizer)** — Find the synthesis

## Interactive Tools

### 1. The Thinking Amplifier
**Type**: Mapper  
**Time**: 60-90 seconds  
**Purpose**: Apply the full 6-layer protocol to any question or decision

**Usage**: Input a question or decision, receive a complete 6-layer analysis

**Location**: `tools/thinking-amplifier.html` or embedded in `validation/index.html`

---

### 2. The Confidence Calibrator
**Type**: Analyzer  
**Time**: 45-60 seconds  
**Purpose**: Run claims or beliefs through epistemic honesty checks

**Usage**: Input a claim, receive confidence calibration with honesty checks

**Location**: `tools/confidence-calibrator.html`

---

### 3. The Perspective Generator
**Type**: Generator  
**Time**: 30-60 seconds  
**Purpose**: Generate three expert perspectives on any situation

**Usage**: Describe a problem, receive perspectives from Execution, Strategy, and Risk experts

**Location**: `tools/perspective-generator.html`

3. **Persona System**: Developed six memorable personas to make abstract layers more accessible

4. **Confidence Calibration**: Integrated epistemic honesty checks into the framework
