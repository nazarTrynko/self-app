# Offline Research Reader (Citation Vault)

**ID:** 119
**Category:** Education
**Tier:** Premium ($10-50)
**APIs:** Files, Share Extension, Local DB
**Offline:** Full

---

## One-Liner

Save PDFs/snippets into an offline citation graph with fast quoting and export.

## Problem

Students and writers lose sources and can’t cite quickly, leading to wasted time and weaker work.

## Solution

Import PDFs and snippets via share sheet, annotate highlights, and export citations + quotes in standard formats.

## Target User

Students, researchers, journalists, long-form writers.

## Key Features

- Import PDFs + web snippets via Share
- Highlight + annotation system
- Citation metadata editor + DOI fields
- Offline search across quotes and notes
- Export: BibTeX/CSV + “quote pack”
