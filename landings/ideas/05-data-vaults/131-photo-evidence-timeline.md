## One-Liner

Turn any photo set into a coherent timeline narrative with notes and dispute-ready exports.

## Problem

Disputes and claims require clear timelines, but albums are messy and lack context.

## Solution

Import photos, add structured notes, generate a timeline narrative, and export a single packet.

## Target User

People dealing with insurance claims, renovations, custody/landlord disputes.

## Key Features

- Import albums + auto chronological grouping
- Timeline note templates (what/where/who)
- Before/after comparisons + side-by-side
- Export narrative packet (PDF + attachments)
- Optional tamper-evident hash log
