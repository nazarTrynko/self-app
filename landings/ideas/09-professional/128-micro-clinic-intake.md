**ID:** 128

---

## One-Liner

Patient intake + consent + visit notes for pop-up clinics and outreach—offline.

## Problem

Mobile clinics rely on paper intake, causing delays, missing consent, and lost data.

## Solution

Offline forms + signatures with encrypted storage and a clean visit packet export for later filing.

## Target User

Pop-up clinics, community health orgs, mobile care providers.

## Key Features

- Intake forms + consent signatures
- Visit notes templates + vitals fields
- Encrypted local storage + device lock
- Export patient packet (PDF)
- Batch export for end-of-day filing

## Monetization

**Model:** Subscription
**Price:** $29.99/mo per device
**Strategy:** Sell via outreach; compliance + offline reliability as core message.
