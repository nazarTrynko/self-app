## One-Liner

Run incidents and drills with protocol checklists, role assignments, and after-action reports—offline.

## Problem

Under stress, teams miss steps and lose a coherent timeline; many “incident” tools depend on connectivity.

## Solution

Choose a playbook, assign roles, run a timed checklist, log actions, and export an after-action report.

## Target User

School admins, event ops, small facilities, volunteer orgs.

## Key Features

- Incident playbooks + drill mode
- Role assignments + responsibilities
- Timed checklists + action log timeline
- After-action report generator (PDF)
- Reusable templates + customization
