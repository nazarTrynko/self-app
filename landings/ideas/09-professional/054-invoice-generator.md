## Technical Notes

**Primary APIs:**
- File System: Invoice and client data storage
- PDF Generation: Professional invoice output
- Contacts: Client import
- Share API: Invoice sending

**Offline Strategy:**
All invoicing data stored locally. PDF generation local. No cloud sync required.

**Data Handling:**
Financial data stored locally with optional encryption. No data leaves device unless exported/shared.

## Competition & Differentiation

**Existing Solutions:** FreshBooks/QuickBooks (subscription, cloud), Wave (cloud, free with upsells)
**Our Edge:** One-time purchase, true offline, data privacy, no subscription overhead

**Complexity:** Medium
**Timeline:** 6-8 weeks
**Key Challenges:** Professional PDF generation, tax calculation flexibility, multi-currency handling
