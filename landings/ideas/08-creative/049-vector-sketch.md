A mobile-first vector illustration app with essential pen tools, layers, shapes, and professional exportâ€”capture design ideas as scalable vectors anywhere, anytime.

## Target User

- Graphic designers sketching concepts
- Logo designers capturing ideas
- Icon designers working on-the-go
- UI/UX designers wireframing
- Artists exploring vector art

## Key Features

- Pen tool with bezier curves
- Shape primitives and boolean operations
- Layer management
- Path editing and manipulation
- Color fills and strokes
- Artboard management
- Export to SVG, PDF, PNG
- Import reference images

## Technical Notes

**Primary APIs:**
- Canvas API / SVG: Vector rendering
- Touch Events: Precise drawing input
- File System: Project storage

**Offline Strategy:**
All rendering and editing local. Projects saved as structured data convertible to standard formats.

**Data Handling:**
All artwork stored locally. Export creates standard vector files.

## Competition & Differentiation

**Existing Solutions:** Vectornator (now Linearity, cloud), Procreate (raster), Adobe Illustrator (desktop)
**Our Edge:** Truly offline, one-time purchase, focused feature set, professional export formats

## Development Estimate

**Complexity:** High
**Timeline:** 12-14 weeks
**Key Challenges:** Bezier curve implementation, touch precision for design work, SVG export fidelity
