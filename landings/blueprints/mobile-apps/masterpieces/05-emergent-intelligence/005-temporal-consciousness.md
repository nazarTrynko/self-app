## Problem

Time is not what clocks measure‚Äîit's what minds experience. A minute of boredom lasts forever; an hour of flow vanishes. Yet we have no tools to perceive our own temporal distortions. They can then intentionally shift temporal experience‚Äîdilating time for learning and performance, compressing it to endure difficulty, and ultimately accessing states where time's grip loosens entirely.

// Time perception is affected by multiple factors
class TemporalState {
  constructor() {
    this.factors = {
      attention: 1.0, // High attention ‚Üí slower time
      emotion: 0.0, // Positive ‚Üí faster, negative ‚Üí slower
      novelty: 0.5, // Novel experiences ‚Üí slower time
      anxiety: 0.0, // Anxiety ‚Üí slower (but unpleasant)
      flow: 0.0, // Flow ‚Üí faster (pleasant)
      fatigue: 0.0, // Fatigue ‚Üí slower
      anticipation: 0.0, // Waiting ‚Üí slower
    };
  }

  // Calculate subjective time multiplier
  // >1 means time feels fast, <1 means time feels slow
  get subjectiveMultiplier() {
    let multiplier = 1.0;

    // High attention slows time
    multiplier -= (this.factors.attention - 0.5) * 0.3;

    // Positive emotion speeds time
    multiplier += this.factors.emotion * 0.2;

    // Novelty slows time
    multiplier -= this.factors.novelty * 0.2;

    // Flow speeds time (but pleasantly)
    multiplier += this.factors.flow * 0.4;

    // Anxiety slows time (unpleasantly)
    multiplier -= this.factors.anxiety * 0.3;

    return Math.max(0.3, Math.min(2.0, multiplier));
  }

  // Present moment "width" in seconds
  get presentWidth() {
    // Focused states have narrower "now"
    // Meditative states have wider "now"
    return 2.0 + (1 - this.factors.attention) * 3.0;
  }
}
