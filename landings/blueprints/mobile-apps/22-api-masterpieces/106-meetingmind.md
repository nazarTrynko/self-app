- Action items get lost in discussion
- Follow-up emails take 15-30 minutes to write
- "What did we decide?" is asked 48 hours later
- Critical context lost when someone misses a meeting

## Key Features

### Core Features (MVP)
- **Universal Recording**: Works for any meeting type
- **Real-Time Transcription**: 95%+ accuracy with speaker labels
- **AI Summary**: Key points, decisions, questions raised
- **Action Item Extraction**: Who, what, when automatically identified
- **Follow-Up Draft**: Ready-to-send email summarizing meeting
- **Calendar Integration**: Auto-attach notes to calendar events
- **Search**: Find anything said in any meeting

### Enhancement Features (v1.5)
- **Meeting Effectiveness Score**: Was this meeting worth it?
- **Speaker Analytics**: Talk time distribution, interruptions
- **Topic Tracking**: See how topics evolve across meetings
- **Smart Scheduling**: AI suggests meeting-free focus blocks
- **Team Workspace**: Share meeting intelligence with team

### Power Features (v2.0)
- **CRM Integration**: Auto-log to Salesforce, HubSpot
- **Task Manager Sync**: Push action items to Asana, Jira
- **Custom AI Prompts**: Extract specific info for your workflow
- **Coaching Insights**: "You talked 80%, try listening more"
- **Meeting Preparation**: AI briefs you before recurring meetings
